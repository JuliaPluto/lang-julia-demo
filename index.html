Both editors share the exact same colorful HighlightStyle — the only difference is the language parser. Because @plutojl/lang-julia builds a full syntax tree, it can tell apart things the line-based legacy mode cannot: function definitions (yellow) vs. calls (cyan), type and module definitions, keyword arguments, macros, integer vs. float literals, string interpolation, regex / raw / byte strings, and every flavor of keyword and operator.
